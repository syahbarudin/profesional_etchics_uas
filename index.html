Checklist Etis Rekayasa Perangkat Lunak

✓

Informed Consent

Pengumpulan data pengguna wajib didasari izin eksplisit dan transparan, bukan manipulasi sistem.

✓

Explainable AI (XAI)

Arsitektur keputusan yang diambil model Machine Learning harus dapat dijelaskan dan diaudit, bukan berupa sistem gelap (Black-Box).

✓

Data Minimization

Hanya memproses variabel data spesifik yang mutlak diperlukan untuk fungsionalitas utama aplikasi.

Kepatuhan Hukum & Regulasi

Pengembangan sistem teknologi wajib selaras secara absolut dengan aturan hukum perlindungan data yang berlaku.

Undang-Undang Pelindungan Data Pribadi (UU PDP)
